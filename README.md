# 🏦 Bank Loan Default Risk Dashboard

An interactive **Bank Loan Default Risk Dashboard** built in **Microsoft Excel** to analyze loan portfolio performance, identify high-risk customer segments, monitor default trends, and support data-driven lending decisions.

---

# 📌 Project Objective

The objective of this project is to transform raw loan application data into meaningful business insights through an interactive Excel dashboard.

The dashboard helps financial institutions:

- Monitor loan portfolio performance
- Identify high-risk borrowers
- Track default trends
- Analyze lending patterns
- Support smarter credit decisions

# 💡 Key Insights

- Overall Default Rate is **24.64%**
- Loan Type 2 has the highest default rate (**34.54%**)
- North region records the highest number of loan applications.
- Applicants aged **above 74 years** represent the highest-risk segment.
- Loan Type 1 contributes the largest loan portfolio.

---

# 📌 Business Recommendations

- Strengthen approval criteria for Loan Type 2.
- Increase verification for applicants aged above 74 years.
- Monitor high-volume regions more closely.
- Improve risk assessment beyond credit score alone.
